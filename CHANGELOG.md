# Changelog

All notable changes to the CAD System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-20

### Added
- Case management system (create, update, close, search, print reports)
- Dispatch system with SLA tracking, auto-assignment, and priority queues
- Evidence management with virtual containers and custody chain
- Forensics module: blood samples, fingerprints, casings, camera traces, toxicology
- EMS dashboard with alerts, blood requests, and unit status tracking
- Fine/ticket system with configurable fine codes
- Jail transfer logging linked to cases
- Security camera placement and live viewing
- News publishing system for in-game press
- Photo system with FiveManage, Medal, Discord, and custom upload providers
- ID reader and vehicle tablet integration
- SolidJS terminal-based NUI with 29+ modal views
- Profile system: simple, full, custom feature sets
- ox_lib rate limiting and job-based access control
- Officer callsign system with duplicate detection
- Public state broadcasting via GlobalState for cases and dispatch
- Offline event queue for reconnecting players
- Config validation on startup with detailed error reporting
- Database auto-schema creation (MySQL via oxmysql)
- Addon system (gcphone integration)
- Localization support (en.json)
- VitePress documentation site
